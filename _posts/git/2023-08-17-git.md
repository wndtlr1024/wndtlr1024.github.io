## repository란

git이 파일버전을 저장해두는 장소를 말한다.<br/>
로컬 작업폴더엔 .git이라는 폴더가있는데 이게 repository(저장소)다.<br/><br/>

실제 개발시 온라인 repository를 많이 사용한다.<br/>
내가 컴퓨터에 만들어 놓은 git repository(저장소)를 온라인에 저장해두는것인데, 이렇게하면 다른 사람과 현업이 가능해진다.
